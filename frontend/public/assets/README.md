Use esta pasta para centralizar todos os assets visuais da loja.

- `icons/`: icones editaveis ou substituiveis
- `logos/`: logo principal, favicon e variacoes
- `headers/`: banners e hero images
- `images/`: imagens gerais da marca
- `images/products/`: imagens dos produtos

Se trocar o nome dos ficheiros, atualize os caminhos em `lib/store.ts` e nos componentes.
